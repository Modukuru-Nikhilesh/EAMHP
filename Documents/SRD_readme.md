# 🛰️ EAMHP – System Requirements Document (SRD)

## 📌 System Overview

EAMHP is a predictive avionics diagnostic tool that analyzes aircraft system parameters to identify and predict faults.

## 🎯 System Purpose

* Automate fault detection
* Assist troubleshooting
* Simulate predictive maintenance

## 📦 System Scope

Includes:

* Parameter input
* Fault prediction
* Diagnostic output

Excludes:

* Real-time aircraft integration
* Live flight data
* Automated repair actions

## ⚙️ Functional Requirements

* Accept user input parameters
* Validate input data
* Analyze system conditions
* Detect faults
* Generate diagnostic outputs
* Suggest faulty LRUs

## 🧩 Non-Functional Requirements

* Fast processing
* Reliable macro execution
* Easy-to-use interface
* Excel compatibility

## ⛔ System Constraints

* Requires Excel macro-enabled environment
* Manual data input required
* Limited scalability
* No external system connectivity
