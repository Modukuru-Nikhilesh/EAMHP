# ✈️ EAMHP – Product Requirements Document (PRD)

## 📌 Overview

EAMHP (Excel-Based Avionics Maintenance & Health Prediction) is a diagnostic support system designed to assist avionics engineers in predicting faults and analyzing aircraft system health using structured parameter inputs.

## 🎯 Product Goal

To provide a lightweight, intelligent, and structured troubleshooting assistant for avionics systems, simulating predictive maintenance logic used in modern aviation environments.

## 👥 Target Users

* Avionics Maintenance Engineers
* Aircraft Maintenance Engineers (AME)
* Aviation Students
* Maintenance Technicians

## 🛫 Supported Aircraft

* Airbus A320 (Initial Version)
* Future: Boeing 737, A320neo

## ⚙️ Core Features

* Parameter-based fault prediction
* LRU fault identification
* System health analysis
* Maintenance recommendations
* Fault probability estimation

## 🔄 Workflow

1. User inputs avionics parameters
2. System processes data using VBA logic
3. Fault prediction engine evaluates conditions
4. Output generated with diagnostics and suggestions

## 📊 Success Metrics

* Accuracy of fault prediction
* Reduction in troubleshooting time
* Usability for engineers
* System reliability

## ⚠️ Limitations

* Rule-based system (no AI yet)
* Limited aircraft compatibility
* Manual data input required
* No real-time aircraft integration
