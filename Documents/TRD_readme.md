# 🧠 EAMHP – Technical Requirements Document (TRD)

## 🏗️ System Architecture

EAMHP uses a macro-driven Excel architecture consisting of input modules, logic engines, and output systems.

## 💻 Platform & Technology

* Microsoft Excel (.xlsm)
* VBA (Visual Basic for Applications)
* Rule-based logic processing

## 🔩 System Components

* Input Module
* Validation Engine
* Logic Engine
* Prediction Engine
* Database Tables
* Output Module

## 📥 Data Inputs

* Voltage parameters
* Sensor outputs
* BIT results
* System status signals

## 🗃️ Database Structure

* Parameter Threshold Tables
* Fault Logic Mapping
* Aircraft Profiles
* Output Logs

## 🔄 System Logic Flow

1. Input received
2. Validation executed
3. Logic engine processes conditions
4. Fault prediction triggered
5. Output generated

## ⚙️ Data Processing

* Threshold comparison
* Rule-based evaluation
* Macro execution
* Result formatting

## ⚠️ Limitations

* Excel scalability constraints
* No real-time data integration
* Limited processing capability

## 🚀 Future Enhancements

* Multi-aircraft support
* Machine learning integration
* Cloud-based data storage
* Predictive analytics engine
