# ✈️ EAMHP – Avionics Health Prediction System

EAMHP (Excel-Based Avionics Mechanical & Health Prediction) is a diagnostic support tool designed to assist avionics engineers in predicting faults and analyzing aircraft system health using structured parameter inputs.

---

## 🚀 Features

* 📊 Parameter-Based Fault Prediction
* ⚙️ LRU Identification
* 📈 Fault Probability Estimation
* 🧠 Rule-Based Diagnostic Logic
* 🛠️ Maintenance Guidance

---

## 🏗️ System Architecture

EAMHP follows a modular architecture consisting of:

* Input Module
* Validation Engine
* Logic Engine
* Fault Prediction Engine
* Probability Calculator
* Output Module

📌 Detailed architecture available in `/architecture`

---

## 🛫 Supported Aircraft

* Airbus A320 (Initial Version)
* Future Support:

  * Boeing 737 Family
  * Airbus A320neo

---

## ▶️ How to Use

1. Open `EAMHP_V1.0.0.xlsm`
2. Enable Macros
3. Enter avionics parameters
4. Run the system
5. View diagnostic outputs
