# ⚙️ Logic Engine

## 📌 Overview

The Logic Engine is the core processing unit of EAMHP, responsible for analyzing validated data using rule-based logic.

## 🎯 Purpose

* Interpret system conditions
* Apply troubleshooting logic
* Detect abnormal behavior

## ⚙️ Functionality

* Executes VBA-based IF conditions
* Compares parameter values with thresholds
* Evaluates multi-parameter combinations

## 📥 Inputs

* Validated parameter data

## 📤 Outputs

* Trigger signals for fault detection
* Intermediate system conditions

## 🔗 Interaction

* Sends processed data to Fault Prediction Engine
* Uses database thresholds and rules

## ⚠️ Limitations

* Static rule-based logic
* No adaptive learning capability
