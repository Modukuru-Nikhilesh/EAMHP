# 📈 Probability Engine

## 📌 Overview

The Probability Engine calculates the likelihood of predicted faults and assigns confidence levels.

## 🎯 Purpose

* Enhance decision-making
* Provide confidence in fault predictions

## ⚙️ Functionality

* Assigns probability percentage (%)
* Generates confidence levels (Low / Medium / High)
* Uses rule weighting or frequency logic

## 📥 Inputs

* Fault predictions from Prediction Engine

## 📤 Outputs

* Probability score
* Confidence index

## 🔗 Interaction

* Sends results to Output Module
* Can integrate with future trend analysis

## ⚠️ Limitations

* Basic estimation (no machine learning)
* Depends on predefined logic
