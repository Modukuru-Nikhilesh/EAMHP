# 📥 Input Module

## 📌 Overview

The Input Module is responsible for collecting avionics system parameters from the user through the Excel interface.

## 🎯 Purpose

* Capture structured aircraft data
* Ensure correct parameter entry
* Serve as the starting point of system processing

## 📥 Inputs

* Voltage values
* Sensor readings
* System status indicators

## ⚙️ Functionality

* Accepts manual user input
* Supports aircraft-specific parameters
* Organizes data into structured format

## 🔗 Interaction

* Sends data to Validation Engine
* Uses predefined parameter fields

## ⚠️ Limitations

* Manual input dependency
* Prone to human entry errors
