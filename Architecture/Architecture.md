# 🏗️ EAMHP Architecture

## 📌 Overview

EAMHP follows a modular Excel-based architecture for avionics fault prediction.

## 🔧 Components

### User Interface

Excel dashboard for input and output.

### Input Module

Handles parameter entry.

### Validation Engine

Ensures correct data input.

### Logic Engine

Processes rule-based conditions using VBA.

### Fault Prediction Engine

Identifies possible faults.

### Probability Calculator

Assigns confidence levels.

### Database Layer

Stores thresholds and logic mappings.

### Output Module

Displays diagnostic results.

## 🔄 Data Flow

1. Input received
2. Validation executed
3. Logic processed
4. Fault predicted
5. Output displayed
